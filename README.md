# 🔬 Undergraduate Research — Linear Algebra & Linear Programming

Research repository for my undergraduate scientific initiation (Iniciação Científica) in **Linear Algebra** and **Linear Programming**.

## About

This repository contains materials, implementations and notes from my undergraduate research conducted at the Federal University of Goiás (UFG), exploring the mathematical foundations that connect classical optimization with quantum computing.

## Topics

- Linear Algebra fundamentals (vector spaces, matrices, eigenvalues)
- Linear Programming and optimization
- Simplex method
- Matrix operations and transformations
- Computational implementations in Python
- Connections between linear algebra and quantum computing

## Tech Stack

- Python
- NumPy / SciPy
- Matplotlib
- Jupyter Lab

## Structure

```
├── notebooks/       → Jupyter notebooks with implementations
├── exercises/       → Problem sets and solutions
├── references/      → Papers and study materials
└── notes/           → Research notes and summaries
```

## Context

Linear Algebra is the mathematical backbone of quantum computing — qubits are vectors, quantum gates are matrices, and measurements are projections. This research builds the mathematical foundation for my work in quantum software development with Qiskit.
